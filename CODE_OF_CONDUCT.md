# Code Of Conduct

This project is published for technical review and focused collaboration.

Expected behavior:

- Be direct, factual, and respectful.
- Keep criticism tied to code, docs, tests, security, or product behavior.
- Do not post secrets, private user data, provider logs, or credentials.
- Do not harass, threaten, or target people.

Maintainers may remove comments, issues, or contributions that violate these
expectations.
