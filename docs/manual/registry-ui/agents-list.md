# Registry UI: Agents list

[← Manual home](../README.md) · [Prev: Dashboard](dashboard.md) · [Next: Agent detail →](agent-detail.md)

**Route:** `/ui/agents` — paginated **cards**, one per enrolled agent. Each card shows **display name**, **role/provider/slug**, a **connectivity** badge, and the latest relative heartbeat. The search box and state dropdown filter the **current page** of results.

**Click** a card to open **agent detail**.
