# Registry UI: Dashboard

[← Manual home](../README.md) · [Prev: Sign in](sign-in.md) · [Next: Agents list →](agents-list.md)

**Route:** `/ui` or `/ui/` — the dashboard is the default landing page after sign-in.

This screen is built from the canonical **`GET /v1/summary`** payload. The hero cards show **connected agents**, **active conversations**, **pending approvals**, **running tasks**, **24h cost**, and **degraded agents**. The lower cards break those same totals into **operations**, **usage**, and **quick-access** links.

Use the dashboard when you want the fastest health check before drilling into agents, conversations, tasks, or usage.
