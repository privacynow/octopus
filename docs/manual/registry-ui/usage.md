# Registry UI: Usage

[← Manual home](../README.md) · [Prev: Skills catalog](skills-catalog.md) · [Next: Deep links →](deep-links.md)

**Route:** `/ui/usage` — **Today**, **7 days**, and **30 days** set `since` / `until` on **`GET /v1/usage`**. You see summary stats and per-conversation rollups **when** the store has usage data; an empty period is normal if nothing has recorded usage-shaped events.
